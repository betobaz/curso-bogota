Windows

Paso 1: Ir a http://www.mongodb.org/downloads

Paso 2: Descargar el archivo de mongo 2.0.6 para su sistema operativo

Paso 3: Descomprimirlo en C:/mongodb

Paso 4: Crear un folder llamado data en C:/ y crear un folder llamado db dentro de ese folder

Paso 5: Abrir aplicacion de linea de comando(tecla de windows, cmd y enter)

Paso 6: Teclear C:/mongodb/bin/mongodb.exe

Para probar que funciono abre otra aplicacion de linea de comando, teclea C:/mongodb/bin/mongodb.exe

Mac

Paso 1: Abrir una terminal

Paso 2: Usar brew para instalar mongodb

brew update
brew install mongodb

Paso 3: Inicar mongodb

mongodb

Para probar que funciono abre otra terminal y corre "mongo"

Tambien pueden usar mac ports

port install mongodb

Ubuntu

Paso 1: Abrir la terminal

Paso 2: Agregar la llave de 10gen a apt

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 7F0CEB10

Paso 3: Agregar el source de 10gen a apt

Abrir /etc/apt/sources.list y agregar 

deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen

Paso 4: Actualizar apt

sudo apt-get update

Paso 5: Instalar mongodb

sudo apt-get install mongodb-10gen

Para probar que funciono

mongo
